Kubos Core BNO055 Sensor API
============================

Enabling this sensor code requires certain configuration values to be present
in the application's configuration json. An example is given below:

::

     {
         "sensors": {
             "bno055": {
                 "i2c_bus": "K_I2C1"
             }
         }
     }
